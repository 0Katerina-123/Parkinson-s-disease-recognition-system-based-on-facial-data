# Parkinson-s-disease-recognition-system-based-on-facial-data
Parkinson's disease recognition system based on facial data
**Краткое описание проекта**

**1. Обработка видео и извлечение признаков**
Видеозаписи мимических проб предварительно обрабатываются: выполняется трекинг лица, извлечение ключевых точек MediaPipe FaceMesh и расчёт геометрических признаков мимики — расстояний, площадей и динамических характеристик движений.

**2. Машинное обучение**
На основе выделенных видеопризнаков формируется обучающая выборка для построения ML-моделей, направленных на классификацию наличия болезни Паркинсона и оценку стадии заболевания по выраженности мимических нарушений.

**3. ЭМГ-валидация видеопризнаков**
Проводится синхронная регистрация видео и поверхностной ЭМГ лицевых мышц. Сопоставление ЭМГ-сигнала с видеопризнаками используется для отбора расстояний между ключевыми точками, наиболее согласованных с реальной активностью целевых мимических мышц.

**4. Веб-сервис**
Разрабатывается программный сервис, реализованный на Python, для загрузки или записи видео, автоматической обработки мимики, расчёта признаков и выдачи результата анализа в удобном пользовательском интерфейсе.
